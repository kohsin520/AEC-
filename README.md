# AEC - Acoustic Echo Cancelling

## What's AEC ?
+ 回音消除
+ 透過`音波干擾`的方式消除自身回傳的聲音
+ 回音由來：用戶A的聲音透過用戶B的喇叭播出，再經用戶B麥克風回傳，導致用戶A會聽見延遲的自身聲音，造成干擾
